# Kanagi image direction

Studio and forge photography in the workshop's own light: directional, slightly
cool daylight from one side, honest material texture. Steel, plaster, kraft
paper, walnut. Never people's faces, never lifestyle scenes.

Prompt template: `{subject}, studio photograph, single directional daylight, plaster and graphite palette with one oxide red accent, honest material texture, shallow depth, quiet composition, photographed for a tool catalog`

Negative prompt: `logos, text, watermark, people, faces, cherry blossoms, calligraphy, neon, vibrant colors, clutter`

Search-query template: `{subject} workshop steel craft macro`

Decision rules:
- `gradient | background | abstract | texture` → AI default
- `product | blade | tool` → stock
- `person | chef | craftsman` → ask
- `building | location` → stock
- `chart | diagram` → AI default
